## Trust Boundaries

| Boundary | Description |
|---|---|
| User numeric input -> local state | Quantity input can be empty, fractional, negative or non-numeric. |
| User clear action -> persistent storage | Clear deletes local list state and requires confirmation. |
| Catalog snapshot -> user-facing metadata | Missing fields must be disclosed without breaking render. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-01-02-01 | Tampering | `QuoteQuantityControl.svelte` / `quoteList.js` | mitigate | Clamp/floor integer quantities and remove below minimum per D-19 and D-20. |

| T-01-02-03 | Spoofing | Item metadata | mitigate | Show missing-data and confirmation badges instead of implying complete stock/code data per D-17 and D-25. |
